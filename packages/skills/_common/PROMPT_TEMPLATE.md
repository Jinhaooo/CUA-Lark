# Skill Prompt 三段式模板

## 模板格式

```
目标：<一句话动作描述，含被参数化的关键值>
区分性描述：<目标元素的相对位置 + 形状/图标特征 + 易混淆邻居及其差异>
完成判据：<可视的、不依赖动画的稳态条件，含被参数化的关键值>
```

## 写作规则

1. **避免绝对像素位置**：使用相对位置（"侧边栏顶部"、"输入框右侧"）+ 形状/图标语义（"放大镜图标"、"灰色 placeholder"）

2. **必须列易混淆邻居**：至少列出1个最容易点错的邻居元素，明确"不要点"

3. **完成判据稳态优先**：避免"点击后0.5s内出现X"这种动态判据，弱VLM难以捕捉瞬态

## 示例

### 搜索联系人

```
目标：在飞书 IM 主面板搜索 "${name_pattern}" 并打开匹配的会话。
区分性描述：
  1. 点击侧边栏顶部的搜索按钮（放大镜图标，位于个人头像下方第一个图标）。
     不要点击：底部的「设置」（齿轮图标）、「联系人」（人形图标）、「会议」（视频摄像机图标）。
  2. 在搜索框输入 "${name_pattern}" 后，会出现下拉结果列表。
     选择"群组"分类下的卡片（左侧是群头像+成员数标记），不要选"消息"分类下的消息片段（左侧是发送者头像+消息预览文本）。
完成判据：
  - 主窗口右侧消息区已切换，会话标题文本完全包含 "${name_pattern}"
  - 输入框、消息列表都已渲染完成（不在 loading 状态）
```

### 发送消息

```
目标：在当前会话发送一条文本消息 "${text}"。
区分性描述：
  1. 输入框位于消息区下方，是一个白色多行可输入区域（带灰色 placeholder "输入消息..."）。
     不要点击：上方工具栏中的"@"按钮、"表情"按钮、"附件"按钮（它们是图标按钮）；
     也不要点击右下角的"语音"或"视频"按钮（它们是圆形图标按钮）。
  2. 用 type 动作输入 "${text}"，并以 "\n" 结尾以提交（spec § 4.1：type content 末尾的 \n 表示发送）。
完成判据：
  - 输入框文本被清空（恢复 placeholder 状态）
  - 消息列表底部新增一条消息气泡，文本完全等于 "${text}"
  - 该气泡靠右显示（自己发出的消息），无红色感叹号或重发按钮
```
